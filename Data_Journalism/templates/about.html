All Parks in NYC

New York City is known to have diverse communities, countless opportunities, and more importantly to this investigation, parks. I wanted to investigate how many parks there were in NYC because no matter where I went, there always seemed to be a park, whether it be big or small, lively or quiet.

It made me think "How common is it to stumble upon a park when you're simply walking?", and then made me begin this investigation.

Cleansing Data

Looking at the data in the CSV file, there was a lot of information that was not very useful to me, such as their jurisdiction area or if they are near a body of water.

Given this, I cut all the data down to only it's most important sections:

  • Park Name
  • Size (in acres)
  • Location
  • Borough
  • Type of Park
  • Zip Code
  • Website Link to NYC Park website about each park

Now that I had the most important data points for my journalism project, I used a python script to convert the CSV file into a JSON file containing nested dictionaries of all the park data, which makes the data a lot more understandable and easier to work with.
